Algorithm - Add two numbers using linked list

Posted on October 25, 2014

You are given two linked lists representing two non-negative numbers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4) Output: 7 -> 0 -> 8

Thinking:

You need a variable to remember sum of each digit.

Handle the situation that one list is longer than the other.

Three variables point to two original lists and one output list.

/** * Definition for singly-linked list. * public class ListNode { * int val; * ListNode next; * ListNode(int x) { * val = x; * next = null; * } * } */

public class Solution { public ListNode addTwoNumbers(ListNode l1, ListNode l2) { int carry = 0; // remember the sum for each digit…………. ListNode newHead = new ListNode(0); ListNode p1 = l1, p2 = l2, p3 = newHead; while(p1!=null ||p2!=null){ if(p1!=null){ carry += p1.val; p1 = p1.next; } if(p2!=null){ carry += p2.val; p2 = p2.next; } p3.next = new ListNode(carry%10); // get mod of carry by 10, save it as the value for p3.next p3=p3.next; carry /=10; } if(carry == 1) p3.next = new ListNode(1); return newHead.next; } }
